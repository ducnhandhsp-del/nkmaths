# LOP TOAN NK Manager

Internal operations app for Lop Toan NK: students, classes, teachers, lessons, attendance, tuition, expenses, and reports.

## Run Locally

```bash
npm install
npm.cmd run dev
```

## Build Checks

Run these before handing off meaningful changes:

```bash
npm.cmd run lint
npm.cmd run build
```

## Repository Map

- `src/` - React/Vite application source.
- `docs/project/` - project status, QA checklist, tasks, and management notes.
- `docs/gas/` - Google Apps Script, Sheet template, and GAS/Sheet contracts.
- `docs/specs/` - focused UI/data specs and implementation notes.
- `docs/archive/` - old prototypes and temporary notes kept for reference.
- `AGENTS.md` and `PROJECT_CONTEXT.md` - current working instructions and product context.
